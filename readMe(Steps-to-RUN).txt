Pharmacy Management System

Setup Instructions:

Step 1: Project Setup
---------------------
1. Create a folder in your desired location.
2. Paste the following files into that folder:
   - PharmacyManagementUI.java
   - medicine_data.csv

Step 2: Open in Code Editor
---------------------------
3. Open the folder in VS Code, IntelliJ IDEA, or any other Java IDE/source code editor of your choice.

Step 3: Install Java
--------------------
4. Make sure Java is installed on your system.
   - If not installed, download and install the JDK from https://www.oracle.com/java/technologies/javase-downloads.html.
   - Verify installation by running:
     java -version

Step 4: Run the Application
---------------------------
5. Compile and run the `PharmacyManagementUI.java` file.
   - For example, in terminal or command prompt, navigate to the project folder and run:
     javac PharmacyManagementUI.java
     java PharmacyManagementUI

Step 5: Use the Application
---------------------------
6. The Pharmacy Management System UI will open.
7. You can **sell medicines** and manage pharmacy stock directly through the UI.

Files Description:
------------------
PharmacyManagementUI.java - Main Java file containing the pharmacy management system with UI.
medicine_data.csv         - Dataset containing medicine details.

Notes:
------
- Ensure both files are in the same folder before running.
- You can update `medicine_data.csv` to add or modify available medicines.
